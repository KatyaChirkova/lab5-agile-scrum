# Лабораторная работа №5 - Agile and Scrum.
Quote Generator - это интерактивное веб-приложение для генерации случайных цитат. Пользователи могут получать вдохновляющие высказывания известных личностей. Приложение предлагает фильтрацию цитат по категориям: философские, юмор, мотивация, технологии и т.д. Реализована возможность сохранять понравившиеся фразы в избранное для последующего просмотра. Проект разработан с использованием современных веб-технологий HTML5, CSS3 и JavaScript. Интерфейс адаптирован для мобильных устройств и имеет оптимизированную производительность.
